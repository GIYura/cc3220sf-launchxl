### Description

LED blinky

### Set up

Target: CC3220SF-LAUNCHXL

### Connection 

| CC3220SF pin | GPIO   | LED        |
|--------------|--------|------------|
| P2 (18)      | P08    | Green Ext  |
| Build-in     | P02    | Green      |
| Build-in     | P01    | Yellow     |
| Build-in     | P64    | Red        |
